# Togetherly
- **Backend:** Java (Spring Boot)
- **Frontend:** React (Vite)

Installation:
### Frontend
- **Node.js 18**: 'https://nodejs.org/en'
- Verify installation : `node -v` `npm-v`
### Backend
- **Java JDK 17+**: `https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html`
- **Maven** : `./mvnw clean install`
- Verify installation : `java -version`

## Environment Setup
1. On terminal, run `git clone https://github.com/haydengan/Project-Hub-Mentorship.git`
2. `cd Project-Hub-Mentorship`
3. Backend Setup:
`cd backend `
`mvn spring-boot:run`
4. Frontend Setup:
`npm install`
`npm run dev`
